# OutputDlS3BackpressureBehavior

Whether to block or drop events when all receivers are exerting backpressure.


## Values

| Name                                  | Value                                 |
| ------------------------------------- | ------------------------------------- |
| `OutputDlS3BackpressureBehaviorBlock` | block                                 |
| `OutputDlS3BackpressureBehaviorDrop`  | drop                                  |
