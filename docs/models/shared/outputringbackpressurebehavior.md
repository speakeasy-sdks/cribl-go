# OutputRingBackpressureBehavior

Whether to block or drop events when all receivers are exerting backpressure.


## Values

| Name                                  | Value                                 |
| ------------------------------------- | ------------------------------------- |
| `OutputRingBackpressureBehaviorBlock` | block                                 |
| `OutputRingBackpressureBehaviorDrop`  | drop                                  |
