# OutputKafkaAcknowledgments

Control the number of required acknowledgments.


## Values

| Name                               | Value                              |
| ---------------------------------- | ---------------------------------- |
| `OutputKafkaAcknowledgmentsOne`    | 1                                  |
| `OutputKafkaAcknowledgmentsZero`   | 0                                  |
| `OutputKafkaAcknowledgmentsMinus1` | -1                                 |
