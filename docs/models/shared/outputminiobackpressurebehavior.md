# OutputMinioBackpressureBehavior

Whether to block or drop events when all receivers are exerting backpressure.


## Values

| Name                                   | Value                                  |
| -------------------------------------- | -------------------------------------- |
| `OutputMinioBackpressureBehaviorBlock` | block                                  |
| `OutputMinioBackpressureBehaviorDrop`  | drop                                   |
