# OutputMskAcknowledgments

Control the number of required acknowledgments.


## Values

| Name                             | Value                            |
| -------------------------------- | -------------------------------- |
| `OutputMskAcknowledgmentsOne`    | 1                                |
| `OutputMskAcknowledgmentsZero`   | 0                                |
| `OutputMskAcknowledgmentsMinus1` | -1                               |
