# OutputGoogleChronicleBackpressureBehavior

| ------------------------------------------------ | ------------------------------------------------ |
| `OutputGoogleChronicleBackpressureBehaviorQueue` | queue                                            |
| `OutputGoogleChronicleBackpressureBehaviorDrop`  | drop                                             |
